# dia_data

This repository contains my analysis of the classic Nokia game **Diamond Rush**. All related assets, tools, and documentation are placed here.

---

## Folder Structure

### 1. **AuroraGT**  
   - This is the **Gameloft Sprite Export Tool**, used for exporting `.sprite` files to `.bsprite` files, this resource is from the Internet.

### 2. **bsprite**  
   - Includes **binary sprite file structures**, this resource is form the Internet.

### 3. **programs**  
   - Stores research-related code, including parsers, scripts, and tools for asset analysis.

### 4. **resources**  
   - Contains **extracted game assets** (images, sounds, maps, etc.).  
   - **If you only need the extracted resources, check this folder directly!**
